## 2. テスト方法

### 手動テスト
1. **リフレッシュ・チェック**: ログイン状態で F5 キーを押し、数秒後に自動的に「同期済み」の緑アイコンが表示されるか確認。
2. **フィルタ・モーダル連動**: カレンダーの役職フィルタを切り替え、「会議体定義から作成」ボタンを押して、リストの中身がフィルタと一致するか確認。
3. **カレンダー画面タスク更新**: カレンダー画面に表示されている未完了タスクをタップし、「完了」に変更して保存。リストが更新され、カレンダーのドットも同期されることを確認。

### 自動ビルド確認
- `npm run build` が正常にパスすること。
