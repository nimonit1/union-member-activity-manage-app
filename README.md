# 役員活動管理アプリ

労働組合の役員活動を効率化するためのタスク・スケジュール管理アプリケーションです。

## 主な機能

- **ダッシュボード**: 今日の予定や未完了タスクを一目で確認。役職に応じたパーソナライズ表示。
- **タスク管理**: カテゴリ別のタスク管理と、組合員対応状況（回答率）の可視化。
- **スケジュール管理**: 会議や出張の予定管理。役職フィルタや進捗（ステータス）管理機能を搭載。
- **役職別管理**: 自分の役職（委員長、書記長など）に合わせたタスク・会議体のみを抽出して表示。
- **定型定義**: 定例の会議体や定型タスクを定義し、ワンクリックで予定・タスクを作成。
- **旅費計算**: 出張経路に基づいた簡便な旅費・交通費の集計。
- **メモ・音声記録**:
    - テキスト、箇条書き、引用、コードブロック、色設定、配置変更等が可能な高度なリッチテキストエディタ。
    - 音声のみのクイックメモにも対応。
    - 保存・閉じるボタンの分離により、編集中に途中保存が可能。
    - PC 版は一覧とエディタの並行表示、スマホ版はフル画面エディタに自動切替。
    - タイトル未入力時の自動採番（`yyyymmdd_nn_森`）機能。
- **PWA 対応**: インストール可能で、オフライン時でも高速に起動・操作。
- **クラウド同期 (Google Drive)**: PCとスマホ間でのデータ共有。`sessionStorage` によりリフレッシュ後の自動再ログインに対応。
- **モバイル対応**: スマートフォンでの操作に最適化された UI/UX。

## 技術スタック

- **Frontend**: React, TypeScript, Vite
- **Styling**: Vanilla CSS, Lucide React (Icons)
- **External API**: Google Drive API (Google Identity Services)
- **Deployment**: GitHub Pages

## 使い方

### 1. 初回セットアップ (Google Drive 同期)
1. 画面上の「Googleでログイン」ボタンをクリックします。
2. Googleアカウントを選択し、アプリ専用フォルダ（`appDataFolder`）へのアクセスを許可します。
3. 同期が完了すると、異なるデバイス間でもデータが自動的に共有されます。
4. **リフレッシュ対応**: トークンはセッション中保持されるため、ページをリフレッシュしても自動的に同期が再開されます。

### 2. 自分の役職を設定する
1. 「設定」メニューから「個人設定」を開きます。
2. 自分の現在の役職を選択して保存します。
3. これにより、自分に関係のある会議体やタスクが優先的に表示されます。

### 3. タスクの管理
- **作成**: タスク画面の「定義から追加」ボタン、または新規作成から追加できます。
- **カテゴリ**: 「組合員対応（🔴）」と「事務タスク（🔵）」を区別して管理できます。
- **進捗更新**: 各タスクの編集画面、または**カレンダー画面の詳細パネル**から「未着手」「進行中」「完了」「保留」を切り替えられます。
- **組合員対応**: アンケートなどの対応状況を「回答率 (%)」として数値管理できます。

### 4. スケジュールの管理
- **作成**: カレンダーの日付を選択し、「会議体から追加」から定例会議を、または「追加」から自由な予定を作成できます。
- **役職フィルタ**: カレンダー上部のフィルタにより、「自分の役職のみ」「全表示」を即座に切り替えられます。
- **進捗（ステータス）**: 予定ごとに進捗を設定できます。完了した予定はカレンダー上で薄く表示され、一目で消化状況がわかります。
- **旅費の紐付け**: 予定の編集画面から、その予定に関連する旅費経路を直接入力できます。

### 5. 旅費の精算
1. 予定の「旅費精算」タブ、または設定のデータ管理から旅費を入力します。
2. 移動経路（出発・到着・金額・往復）を入力すると自動で集計されます。

## 開発者向け情報

### ローカル開発
1. `npm install`
2. `npm run dev`

### クラウド同期設定
Google Cloud ConsoleでOAuth 2.0 クライアントIDを取得し、`src/utils/googleDrive.ts` の `CLIENT_ID` を設定してください。
- 承認済みのJavaScript生成元: `http://localhost:5173`, `https://[your-username].github.io`
