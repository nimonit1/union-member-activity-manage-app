# TeraQ (役員活動管理アプリ)

労働組合の役員活動を効率化するためのタスク・スケジュール管理アプリケーションです。

## 主な機能

- **ダッシュボード**: 今日の予定や未完了タスクを一目で確認。役職に応じたパーソナライズ表示。
- **タスク管理**: カテゴリ別のタスク管理と、組合員対応状況（回答率）の可視化。
- **スケジュール管理**: 会議や出張の予定管理。
- **役職別管理**: 自分の役職（委員長、書記長など）に合わせたタスク・会議体のみを自動抽出。
- **定型定義**: 定例の会議体や定型タスクを定義し、ワンクリックで予定・タスクを作成。
- **旅費計算**: 出張経路に基づいた簡便な旅費・交通費の集計。
- **クラウド同期 (Google Drive)**: PCとスマホ間でのデータ共有機能。
  - Google Driveのアプリ専用領域 (`appDataFolder`) を使用し、安全にデータを同期します。
  - ページ更新時の自動再接続機能を搭載。
- **モバイル対応**: スマートフォンでの操作に最適化されたヘッダーとナビゲーション。

## 技術スタック

- **Frontend**: React, TypeScript, Vite
- **Styling**: CSS (バニラCSS), Lucide React (アイコン)
- **External API**: Google Drive API (Google Identity Services)
- **Deployment**: GitHub Pages

## 使い方

### 1. 初回セットアップ (Google Drive 同期)
1. 画面下部（モバイルはヘッダー）の「Googleでログイン」ボタンをクリックします。
2. Googleアカウントを選択し、アプリ専用フォルダへのアクセス（appDataFolderへの権限）を許可します。
3. 同期が完了すると、異なるデバイス間でもデータが自動的に共有されるようになります。

### 2. 自分の役職を設定する
1. メニュー（サイドバー）の「設定」をクリックします。
2. 「個人設定」のドロップダウンから自分の現在の役職（例: 委員長）を選択します。
3. 設定を保存すると、ダッシュボード、タスク一覧、カレンダーがその役職に関連するものだけにフィルタリングされます。
   - ※ 他の役職のタスクも見たい場合は、設定で「全表示（例外対応）」をONにしてください。

### 3. 会議やタスクの作成
- **カレンダー**: 日付を選択し、「会議体から追加」ボタンを押すと、定義済みの定例会議を一括選定できます。
- **タスク一覧**: 画面右上のテンプレート一覧から、よくある活動（調査、聞き取り等）をワンクリックで追加できます。

### 4. 旅費の管理
1. カレンダーの予定入力画面で「旅費を入力」をクリックします。
2. 移動経路（出発・到着・金額・往復）を入力します。
3. 入力した内容は自動的に集計され、年間の活動経費として把握できます。

### 5. データのバックアップ・エクスポート
1. 「設定」画面の「個人設定・データ管理」セクションを開きます。
2. 以下のボタンからデータを JSON 形式でエクスポート（ダウンロード）できます。
   - **設定のみを保存**: 役職定義、定型タスク、会議体定義のみを保存します（他端末への設定引き継ぎに便利）。
   - **全データを保存**: すべてのタスク、予定、設定を保存します（完全なバックアップ用）。

## 開発者向け情報

### ローカル開発
1. `npm install` で依存関係をインストール。
2. `npm run dev` でローカル開発サーバーを起動。

### クラウド同期設定
クラウド同期を利用するには、Google Cloud ConsoleでOAuth 2.0 クライアントIDを取得し、`src/utils/googleDrive.ts` の `CLIENT_ID` を差し替えてください。
- 承認済みのJavaScript生成元: `http://localhost:5173` (開発時), `https://[your-username].github.io` (本番時)
